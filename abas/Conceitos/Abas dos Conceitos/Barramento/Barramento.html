Barramento

VÍDEO EM LIBRAS

CONCEITO

É um longo pedaço do circuito que serve para aproximar vários pontos do circuito para conectar todos juntos. Na grande maioria dos casos o barramento é usado para a alimentação do circuito, facilitando fornecer energia para todo o circuito.

EXEMPLO

Nas protoboards, os barramentos são as linhas horizontais usadas para a alimentação, geralmente com uma linha vermelha para representar o VCC e outra preta ou azul para o GND. As protoboards podem ter vários barramentos, o que facilita ainda mais as conexões e permite a criação de vários circuitos completamente diferentes e separados uns dos outros.
